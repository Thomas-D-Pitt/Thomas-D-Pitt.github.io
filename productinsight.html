Product Insight

While working at Product Insight the majority of my time was spent creating simulation and control software for a 3.5 axis gantry robot. This includes creating a GUI for manually controlling the robot as well as designing automated control software. I also worked with a team to create a simulation of the robot capable of acting as a drop in replacement of the physical robot. The simulation received an executed commands mirroring how the physical robot would behave given the same commands. The simulation also provided graphs displaying position, velocity, and estimated power consumption
Other things I've done while working at Product Insight include developing software to manage a 5 servo motor system over a CANopen network, and developing STM32 firmware acting as a CAN interface for a network connected computer. I also included a modified bootloader capable of flashing new firmware over the same network connection.
