Sailbot

This project implements the Paxos algorithm on top of a chat program that allows users to create chat rooms and talk with others (also made by me). A key part of system design is the inability to be both highly available and highly consistent at the same time. This project is an implementation of a highly consistent algorithm called Paxos which uses a series of consensus problems to guarantee all servers remain consistent is the presence of networking errors and server crashes. This algorithm is used in many production systems that need to ensure data is identical across many servers, some examples of services that use this algorithm are Bigtable and Chubby by Google, Google Spanner, Autopilot Cluster management by Microsoft, Amazon DynamoDB and more
